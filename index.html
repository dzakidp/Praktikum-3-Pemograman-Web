1. Penjelasan Konsep Dasar PHP

a. Jelaskan apa itu PHP!

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi berbasis web...

b. Sebutkan keunggulan PHP dibandingkan dengan bahasa pemrograman lainnya!

  • Open Source
  • Kompatibilitas yang Luas
  • Dukungan Basis Data yang Luas
  • Kemudahan Belajar
  • Kinerja Cepat

2. Perbandingan GET dan POST

a. Jelaskan perbedaan antara metode GET dan POST!

GET: Data dikirimkan melalui URL sehingga terlihat oleh pengguna...

POST: Data dikirimkan melalui body HTTP sehingga lebih aman dibanding GET...

b. Contoh kasus penggunaan:

  • GET: Pencarian produk pada situs e-commerce.
  • POST: Formulir login atau pendaftaran.

3. Penggunaan include, require, dan koneksi PHP dengan MySQL

a. Jelaskan fungsi include dan require!

Include: Memasukkan file eksternal ke dalam skrip PHP...

Require: Sama dengan include, tetapi eksekusi program akan dihentikan jika file tidak ditemukan...

b. Contoh koneksi PHP dengan MySQL:

<?php
$servername = "localhost";
$username = "root";
$password = "";
$database = "exampleDB";

$conn = new mysqli($servername, $username, $password, $database);

if ($conn->connect_error) {
    die("Koneksi gagal: " . $conn->connect_error);
}
echo "Koneksi berhasil";
?>
